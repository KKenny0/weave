# weave

The primary project README is maintained in Chinese.

[Read the Chinese README](README.md)

[Migration from Loom](MIGRATION.md)

weave is an Agent Skill that turns source bundles, technical projects, and research domains into evidence-grounded Chinese longform articles. It defines an observable reading capability before research, builds a route-specific evidence model, compares candidate frames, tests the selected frame against held-out material, and runs reconstruction, novel-case, counterexample, and question-repair probes before composition. Internal research and reader-model artifacts stay out of the finished article.
